Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le World Wide Web (WWW ou simplement le Web) est un système d'information qui permet le partage de contenu sur Internet par le biais de moyens conviviaux destinés à attirer des utilisateurs au-delà des spécialistes et des amateurs de l'informatique .

Internet

Ensemble de réseaux mondiaux interconnectés qui permet à des ordinateurs et à des serveurs de communiquer efficacement au moyen d'un protocole de communication commun (IP).

Cloud

Ensemble des serveurs distants proposant des services accessibles par le réseau Internet.

W3C

Organisation Internationale de normalisation qui définit les standards ouverts du web. Ce sont les normes de developpement du Web.

Partie 2 - Distinguez les langages du Web

Langages client

C'est un langage de programmation exécuté directement par le navigateur web de l'utilisateur, permet de rendre une page interactive, généralement du HTML, CSS ou JavaScript.

Navigateurs

Le navigateur web est un logiciel installé sur l'ordinateur ou le smartphone qui permet d'afficher des pages et applications du Web.

Langages serveur

C'est un langage de programmation utilisé pour exécuter des tâches serveur web, par opposition au client (navigateur). On aura par exemple le PHP, Python, Java...

Frameworks

Les Frameworks web sont des ensembles d'outils, bibliothèques qui simplifient le developpement web, interface et gestion des données. Ils accélèrent la création des sites dynamiques.

Bases de données

C'est un système organisé pour stocker, gérer et récupérer des données utilisées par les applications web.

Site responsive

Un site responsive est un site web conçu pour s'adapter automatiquement à différents appareils et taille d'écran grace à des techniques comme le CSS ou les framesworks front-end.

Application mobile

C'est une application conçue pour fonctionner sur des appareils mobiles (smartphones, tablettes) avec des langages ou frameworks, ou des solutions multiplateformes comme React Native.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

C'est un ordinateur spécialement conçu pour gérer, stocker, traiter et distribuer des informations, données et services au sein d'un réseau informatique. Contrairement à un ordinateur personnel classique, un serveur est optimisé pour des tâches spécifiques.

Adresse IP

Une adresse IP est un nombre de 32 bits. Il identifie de manière unique un hôte (ordinateur ou autre appareil, tel qu'une imprimante ou un routeur) sur un réseau TCP/IP. Les adresses IP sont généralement exprimées au format décimal en pointillés, avec quatre nombres séparés par des points, tels que 192.168.1.1.

Nom d’hôte et DNS

Le nom attribué comprend le nom de l'hôte de l'ordinateur (c'est à dire de l'ordinateur dans les propriétés du systéme) et le nom DNS (Domain Name System) du domaine Active Directory rejoint par l'ordinateur

Protocoles

Dans le domaine des réseaux, un protocole est un ensemble normalisé de règles règissant le formatage et le traitement des données. Les protocoles permettent aux ordinateurs de communiquer entre eux.

Partie 4 - Identifiez les métiers du web

Développeurs

Le développeur, également appelé programmeur, est un informaticien qui oeuvre auprés des utilisateurs pour concevoir des logiciels adaptés à leurs besoins. De l'analyse à l'application en passant par la programmation, il connaît tous les rouages pour mettre au point des programmes informatiques sur mesure.

Product Managers et chefs de projets

Le Product Manager, ou chef de produit, est responsable de la gestion du cycle de vie complet d'un produit. Il supervise le développement de la conception jusqu'a la commercialisation, en passant par la phase de prototypage, les essais et la fabrication. Le Chef de projets définit, coordonne et suprervise toutes les étapes d'un projet de sa conception au déploiement de la solution et en assurant une formation auprès des futurs utilisateurs.

Designers

Le Designer définit la forme, le comportement des produits et des services numériques. Il est le designer industriel du domaine numérique. Son travail consiste en la création de produits (site web, logiciels, objets interactifs,...) ou de services nultimédias, en y intégrant du texte, des images, des sons.

Testeurs

Le Testeur est le spécialiste de la chasse aux bugs, ces erreurs qui empêchent le bon fonctionnement d'un logiciel. A Lui de les signaler au service développement.

Techniciens informatique

Aussi appelé technicien de maintenance en informatique, c'est un personnage clé dans la plupart des entreprises du tertiaire, il est en charge de la configuration et la maintenance des ordinateurs, serveurs, routeurs et autres matériels informatiques. Le diagnostic et la résolution des problèmes complexes liés au matériel ou logiciel. L'installation et la mlise à jour de logiciels et systèmes d'exploitation font partie de ses fonctions.

Langages de développement web

HTML

Front-End

Squelette du site Web. Il définit la structure de base du contenu, comme les titres, les paragraphes, les images, etc.

CSS

Front-End

Le CSS (Cascading Style Sheets ou feuilles de style en cascade) représente un langage informatique dédié à la mise en forme visuelle des documents web.

JavaScript

Front-End

JavaScript est un langage de programmation utilisé par les développeurs pour concevoir des sites web interactifs

JavaScript

Back-End

Les fonctions JavaScript peuvent permettre d'améliorer l'expérience utilisateur d'un site web, de la mise à jour des flux de médias sociaux à l'affichage d'animations et de cartes interactives.

PHP

Back-End

PHP (officiellement, ce sigle est un acronyme récursif pour PHP Hypertext Preprocessor) est un langage de scripts généraliste et Open Source, spécialement conçu pour le développement d'applications web. C'est un language coté Serveur pour le Web, Gratuit et simple a prendre en main.

Java

Back-End

Java est un langage de programmation à usage général, évolué et orienté objet dont la syntaxe de haut niveau, orienté programmation. Il n'est pas opensource.

Python

Back-End

Python est un langage de programmation orienté objet facile à interpréter et de haut niveau avec une syntaxe facile à lire.C'est un language très utilisé pour l'IA de nos jours. Il très utilisé aussi pour le machine learning.

C#

Back-End

C# est un langage de programmation orientée objet, fortement typé, dérivé de C et de C++, ressemblant au langage Java

SQL

Bases de données

SQL (Structured Query Language) est un langage de requête populaire fréquemment utilisé dans tous les types d'applications. Il est très utilisé pour interroger des bases de données.

Boîte à outils du développeur

  • IDE comme VS Code : Visual Studio Code (VSCode) est un éditeur de code source et un environnement de développement intégré (IDE) de Microsoft.
  • Github : GitHub est une plateforme basée sur le cloud où vous pouvez stocker, partager et travailler avec d'autres pour écrire du code. Platforme Git.
  • Librairies / bibliothèques : C'est, en apparence, comme une vraie bibliothèque pour le développeur : il peut y puiser tel ou tel ouvrage, selon ses besoins. Par exemple bibbliothèques de boites de dialogues.
  • Frameworks : Un framework est un ensemble d'outils et de composants logiciels organisés conformément à un plan d'architecture et des patterns, l'ensemble formant ou promouvant un « squelette » de programme, un canevas.On aura par exemple : AngularJS.... VueJS. ... Ruby on Rails.
